# 羽升 AIMFlly - Personal Website & Growth Archive

[![Next.js](https://img.shields.io/badge/Next.js-App_Router-black?logo=next.js)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue?logo=typescript)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-v4-38B2AC?logo=tailwind-css)](https://tailwindcss.com/)

本项目是**羽升 AIMFlly** 的个人主页与技术成长记录平台。它不仅用于展示个人作品与项目集，更是从零开始探索现代前端工程化的“活文档”与实践沙盒。

## 📖 项目简介

本项目旨在构建一个兼具卓越性能、现代美学与流畅交互体验的 Web 站点。通过不断的迭代与重构，项目展示了现代 UI 设计理念的落地以及前端动效（如全局导航、微交互）的工程化实现。

## ✨ 核心特性

- **现代化前端架构**：基于 Next.js App Router 构建，提供极致的渲染性能。
- **模块化设计体系**：采用 Tailwind CSS v4 与 CSS-first 理念，实现流式响应式与高度一致的视觉语言。
- **丝滑的交互体验**：融入微动效、系统级毛玻璃（Glassmorphism）与符合直觉的用户体验设计。
- **严格的工程规范**：内置 ESLint，并遵循详尽的代码与设计规范，保障项目长期可维护性。

## 🏛️ 项目历史与 Archive (Vibe Coding 纪念碑)

在工程的演进过程中，我们在以下目录保留了早期的实验性代码：

- `docs/单页HTML/source/`
- `docs/单页HTML/其他页面/`

**背景说明**：
这是最初接触 **Vibe Coding（AI 辅助编程）** 阶段的早期产物。在当时的认知下，曾具有“**单页 HTML 可以实现世界上所有 Web 项目**”的技术预设。随着对前端工程化、组件化及构建工具的深入理解，这一认知已被现代开发模式所彻底替代。

特此保留这些原生的单页 HTML 文件，旨在将其作为项目初创期的“纪念碑”，以自嘲与幽默的方式，记录技术认知的演变与个人成长的轨迹。

## 🚀 快速开始

请确保您的本地环境已安装 [Node.js](https://nodejs.org/) (推荐版本 18.x 或更高)。

```bash
# 1. 下载或克隆本项目代码
# git clone <repository-url>

# 2. 进入项目根目录并安装依赖
npm install

# 3. 启动本地开发服务器
npm run dev
```

启动完成后，请在浏览器中访问 [http://localhost:6190](http://localhost:6190) 预览项目。
