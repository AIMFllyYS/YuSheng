📘 第四节

微积分基本定理

把"定积分"和"导数/原函数"连起来——这是微积分的核心桥梁

SECTION 01

积分函数 F(x)

设 $f$ 在 $[a,b]$ 上黎曼可积。对每个 $x \in [a,b]$，定义：

$$F(x) = \int_a^x f(t)\,dt$$
三个要点：
  • 积分上限是变量 $x$，下限 $a$ 是固定常数
  • 积分号里用 $t$（哑变量），避免与上限 $x$ 冲突
  • 对每个 $x$，这个定积分都是一个数 → $F$ 是一个新函数

物理直观：$F(x)$ 表示"从 $a$ 累积到 $x$ 的总量"。比如 $f(t)$ 是速度，$F(x)$ 就是从 $t=a$ 到 $t=x$ 走过的路程。

SECTION 02

积分函数一定连续

📜 定理 A：积分函数的连续性

若 $f$ 在 $[a,b]$ 上可积，则 $F(x) = \int_a^x f(t)\,dt$ 在 $[a,b]$ 上连续。

证明思路：看 $F$ 在两点之间相差多少。

$$|F(x) - F(x_0)| = \left|\int_{x_0}^x f(t)\,dt\right| \leq M|x - x_0|$$

其中 $M$ 是 $|f|$ 的上界。当 $x \to x_0$ 时，右边趋于 0，所以 $F$ 连续。

直觉理解：累积量怎么可能突然跳断？多走一点点区间，只会多一点点面积。
SECTION 03

微积分基本定理 I（FTC I）

⭐ 微积分基本定理 I

若 $f$ 在 $[a,b]$ 上连续，且 $F(x) = \int_a^x f(t)\,dt$，则 $F$ 在 $(a,b)$ 上可导，且：

$$F'(x) = f(x)$$

一句话：面积函数的导数 = 当前的高度

证明核心：考虑差商

$$\frac{F(x+h) - F(x)}{h} = \frac{1}{h}\int_x^{x+h} f(t)\,dt$$

右边是 $f$ 在 $[x, x+h]$ 上的平均值。当 $h \to 0$ 时，由于 $f$ 连续，平均值趋近于 $f(x)$。

SECTION 04

牛顿–莱布尼兹公式（FTC II）

⭐ 微积分基本定理 II（牛顿–莱布尼兹公式）

若 $f$ 在 $[a,b]$ 上连续，$F$ 是 $f$ 的任意一个原函数（即 $F'(x) = f(x)$），则：

$$\int_a^b f(x)\,dx = F(b) - F(a)$$

常记作：$\int_a^b f(x)\,dx = F(x)\Big|_a^b = F(b) - F(a)$

这就是你以后算定积分几乎唯一用的公式！

SECTION 05

不定积分与定积分的关系

📐 不定积分

若 $F'(x) = f(x)$，则 $F$ 是 $f$ 的一个原函数。全体原函数记作：

$$\int f(x)\,dx = F(x) + C$$

这是一个函数族，不是一个数。

核心关系：

定积分 = 原函数在端点的差

先求不定积分 → 再代入上下限相减

📊 三者关系：f, F, ∫

SECTION 06

例题演练

📝 例 1：多项式积分

求 $\displaystyle\int_0^1 x^2\,dx$

解：原函数 $F(x) = \frac{1}{3}x^3$

$$\int_0^1 x^2\,dx = \frac{1}{3}x^3\Big|_0^1 = \frac{1}{3}(1^3 - 0^3) = \frac{1}{3}$$
📝 例 2：三角函数积分

求 $\displaystyle\int_0^\pi \sin x\,dx$

解：原函数 $F(x) = -\cos x$

$$\int_0^\pi \sin x\,dx = -\cos x\Big|_0^\pi = (-\cos\pi) - (-\cos 0) = 1 + 1 = 2$$
📝 例 3：分段函数积分

设 $f(x) = \begin{cases} 1, & 0 \leq x < 1 \\ 2, & 1 \leq x \leq 3 \end{cases}$，求 $\int_0^3 f(x)\,dx$

解：利用区间可加性分段计算：

$$\int_0^3 f(x)\,dx = \int_0^1 1\,dx + \int_1^3 2\,dx = 1 + 4 = 5$$
