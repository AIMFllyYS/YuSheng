📘 第六节

定积分的几何意义与面积应用

带符号面积、真实面积、两曲线间面积——把"积分=累积量"的几何直观彻底盘清楚

SECTION 01

定积分算的是"带符号的面积"

核心概念：

定积分 $\int_a^b f(x)\,dx$ 计算的是曲线 $y=f(x)$、区间 $[a,b]$、x 轴围成图形的带符号面积。

📐 符号规则
  • $f(x) \geq 0$：曲线在 x 轴上方，面积记为正
  • $f(x) \leq 0$：曲线在 x 轴下方，面积记为负
  • 定积分 = 正面积 − 负面积（净效果）
📜 两种面积的计算

带符号面积（净量）：位移、净电荷、净热量…

$$\text{净量} = \int_a^b f(x)\,dx$$

真实几何面积（总量）：路程、总流量、总消耗…

$$\text{实际面积} = \int_a^b |f(x)|\,dx$$
📝 物理类比

速度 $v(t)$：正 = 向前，负 = 向后

  • $\int v\,dt$ = 位移（正负会抵消）
  • $\int |v|\,dt$ = 总路程（都当正的加）

SECTION 04

图例 3：两曲线 y = x 与 y = x² 之间的面积

📜 两曲线间面积公式

若 $f(x) \geq g(x)$ 在 $[a,b]$ 上成立，则两曲线 $y=f(x)$、$y=g(x)$ 与直线 $x=a$、$x=b$ 围成的面积：

$$S = \int_a^b [f(x) - g(x)]\,dx = \int_a^b (\text{上} - \text{下})\,dx$$

📊 y = x 与 y = x² 的夹面积

y = x（上方）
y = x²（下方）
夹面积
∫₀¹ x dx
1/2
∫₀¹ x² dx
1/3
夹面积
1/6
📝 计算过程

在 $[0,1]$ 上，$x \geq x^2$（因为 $0 \leq x \leq 1$ 时平方会变小）

$$S = \int_0^1 (x - x^2)\,dx = \frac{1}{2}x^2\Big|_0^1 - \frac{1}{3}x^3\Big|_0^1 = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}$$
直观理解：

对每个 x，竖着画一条线段，上端在 $f(x)$，下端在 $g(x)$，高度 = $f(x) - g(x)$。把这些无数个小竖条的面积累积起来，就是整个夹区域的面积。

SECTION 05

分段换号的面积处理

当函数穿过 x 轴时，需要按零点分段，分别处理正负区域。

📝 例题：y = x - ½ 在 [0,1] 上的面积

零点：$x - \frac{1}{2} = 0 \Rightarrow x = \frac{1}{2}$

  • $[0, \frac{1}{2}]$：$f(x) < 0$（在 x 轴下方）
  • $[\frac{1}{2}, 1]$：$f(x) > 0$（在 x 轴上方）

📊 y = x - ½ 分段处理

负区域 [0, ½]
正区域 [½, 1]
左半（负）
-1/8
右半（正）
+1/8
净面积 ∫f
0
实际面积 ∫|f|
1/4
📜 计算实际面积的方法
$$S = \int_0^{1/2} \left|\,x - \frac{1}{2}\,\right|dx + \int_{1/2}^1 \left|\,x - \frac{1}{2}\,\right|dx$$

在左边：$|x-\frac{1}{2}| = \frac{1}{2} - x$

在右边：$|x-\frac{1}{2}| = x - \frac{1}{2}$

$$S = \int_0^{1/2} \left(\frac{1}{2}-x\right)dx + \int_{1/2}^1 \left(x-\frac{1}{2}\right)dx = \frac{1}{8} + \frac{1}{8} = \frac{1}{4}$$
