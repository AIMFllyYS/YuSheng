# UI与设计规范

本项目的设计理念侧重于“每个像素都有其目的，克制即奢华”，以用户心理学为核心驱动设计决策。

## 1. 约束与受众驱动

在进行任何组件或页面的设计前，需明确**时间线**、**受众群体**、**品牌约束**和**内容结构**。
- **目标受众导向**：例如针对 Gen Z 设计应偏向大胆与移动优先，而面向 B2B 用户则需要提供专业感与信任感。
- **敢于突破**：拒绝刻板的“AI感设计”（如泛滥的紫色、默认暗黑加霓虹效果、无理由的 Bento 布局或毛玻璃效果），寻找最适合品牌的设计表达。

## 2. 交互心理学与可用性 (UX Psychology)

- **Hick's Law (席克定律)**：减少用户的选择数量，采用渐进式披露以降低决策成本。
- **Fitts' Law (菲茨定律)**：增加重要交互元素（如主按钮）的可点击区域（尺寸与边距）。
- **Miller's Law (米勒定律)**：人类工作记忆容量有限，将内容或选项分组（区块化），每组约 5-7 项。
- **Von Restorff Effect (孤立效应)**：使主要 CTA（行动号召按钮）在视觉上明显区别于其他次要元素。

## 3. 布局与栅格原理

- **黄金分割比 (1.618)**：用于大层面的比例设计（如主内容与侧边栏 62%:38%，字号梯度递增）。
- **8点网格系统 (8-Point Grid)**：所有的内外边距、宽高尺寸均使用 8 的倍数（如 8px, 16px, 24px, 32px等），极小间距可用 4px。
- **可读性约束**：文本段落单行控制在 45-75 个字符之间，行高通常保持 1.4 - 1.6。

## 4. 色彩与视觉层级

- **60-30-10 法则**：
  - **60%** 作为主导色/背景色（通常为平和的中性色）。
  - **30%** 作为辅助色。
  - **10%** 作为强调色（CTA、警告或核心亮点）。
- **情感映射**：根据产品表达的情感（信任选蓝、自然选绿、奢侈选深青/金等）合理选用色彩族，而非盲目套用。

## 5. 动画与特效原则

- **动画耗时**：距离越远、尺寸越大，动画耗时越长。关键动作需要快而明确。
- **缓动曲线 (Easing)**：
  - **入场**：`ease-out`（减速，平稳落座）
  - **出场**：`ease-in`（加速离开）
  - **强调**：`ease-in-out`（平滑流畅）
- **性能优化**：主要只针对 `transform` 和 `opacity` 进行动画，并尊重系统的“减弱动态效果” (reduced-motion) 设置。
- **玻璃态 (Glassmorphism)** 与 **阴影层次** 需适度使用，保持多层级阴影的真实感。
